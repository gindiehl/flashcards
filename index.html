An object-oriented computer programming language commonly used to create interactive effects within web browsers.

noun
An operator is used to assign values, compare values, perform arithmetic operations, and more.
some examples: + - * /

noun
Variables are containers for storing data values.

noun
In computer programming, a naming convention is a set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities in source code and documentation.

noun
A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).
example:
function myFunction(p1, p2) {
return p1 * p2; // The function returns the product of p1 and p2
}

noun
The arguments object is a local variable available within all functions. You can refer to a function's arguments within the function by using the arguments object. In JavaScript, a function may be called with any number of arguments, no matter how many of them are listed.
example:
var name = function(first,last) {
alert(first + " " + last);
}
In this example "first" and "last" are both arguments to the name function. When we call the function like this: name("Sean","Pierce"); an alert will pop up concatenating the "first" and "last" argumanets to say "Sean Pierce"

noun
A parameter in Javascript is similar to an argument in that it is passed within the parentesis of a function. ex: name(param1,param2)
